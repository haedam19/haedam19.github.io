# 파일 생성
_post 폴더에 yyyy-mm-dd-{title}.md 형식의 이름으로 파일 작성. title은 영문으로.

# 머릿말 작성법
title : 제목. 큰 따옴표로 감싼 문자열로 작성. 작성하지 않을 경우 파일명에 있는 title을 제목으로 사용.  
excerpt : 포스트 목록에서 보여지는 요약글  
toc : table of contents  
toc_label : toc 이름  
categories : 카테고리  
tags : 태그  
date : 작성 날짜  
last_modified_at : 마지막 수정 날짜
